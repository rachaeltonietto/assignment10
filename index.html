Falling for JavaScript

Rachael Tonietto

In this week's lesson, I learned how to adapt existing JavaScript to my own work. In this case, this adaptation allowed for my images to fade unless a mouse is hovered over them. I wrote my own CSS to edit the look of the page, as well as used CSS provided by bootstrap. I used the bootstap grid system to make my images float to the left and stack on top of one another.
